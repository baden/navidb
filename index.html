NaviDB is a library to be dropped into any other Erlang project, to be used to assist DevOps people diagnose problems in production nodes.

The source code can be obtained from the github repo.

Included modules are:

navidb
Main module, contains basic functionality.
navidb_subs
Subscribe to resource changes.
